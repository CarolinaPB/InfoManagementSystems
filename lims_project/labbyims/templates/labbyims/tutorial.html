Labby tutorial



Welcome to the Labby tutorial. Here we will explain how Labby works and how you can best use it to improve your laboratory management!





The front page



Above is the front page. The numbered parts are as follows:


  1. The search bar. Using the drop-down menu, you can choose between searching for a particular product unit, a location or product.
  2. The reminders. Here you will see reminders about reservations you have made, product units that are about to expire or need to be retested and product units that are running out. Every blue column title can be clicked to sort by that attribute.
  3. The top bar. It has several options and is accessible from every page of the site. First is a simple search bar, this one only allows searches for product units. Second are three buttons: Add product, Add item and Update item. These link to their respective pages, see below. Third is the user drop down menu. Here you can find links to various other sites, see below.




Above is the content of the drop-down menu. The links lead to different parts of the site as well as allowing the user to log out.





The add product page



On this page, you can add new products to the database. A product must have a CAS number and a (descriptive) name. You must also define the temperature range in which it should be kept and any constraints it might have, such as it being flammable, to ensure it is only stored in a compatible location.
You can also head straight to the add product unit page by clicking the "Add unit" link.





The add item CAS page



When the add item button is clicked, it redirects you to this page. Here you must enter the CAS number of a product in the database. It will then take you to the add product unit where you can add a unit of that product to the database.





The add product unit page





Once the CAS number has been chosen, you get to the add product unit page. On this page you can add a new container of a particular chemical. You can change the product this unit is a part of, in case you chose the wrong CAS number. In order to create a new unit, you must input a description (we recommend something descriptive), a delivery date, a company that produced it, an initial amount as well as the measurement unit and finally a location.
All other attributes on this page are optional or have defaults but we recommend that you associate any product unit with at least one department. This allows users associated with this department to watch it's status, getting reminders if it is close to finished. If you intend to add several units with exactly the same attribute values, this is also possible. Simply input the number of units to add at the bottom field.





The add product unit page



If a product unit has been moved, it has been opened or some amount used, you can use the update product unit page to update the information for that unit.
