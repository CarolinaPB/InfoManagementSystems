Labbi tutorial



Welcome to the Labbi tutorial. Here we will explain how Labbi works and how you can best use it to improve your laboratory management!





The front page



Above is the front page. The numbered parts are as follows:


  1. The search bar. Using the drop-down menu, you can choose between searching for a particular product unit, a location or product.
  2. The reminders. Here you will see reminders about reservations you have made, product units that are about to expire or need to be retested and product units that are running out. Every blue column title can be clicked to sort by that attribute.
  3. The top bar. It has several options and is accessible from every page of the site. First is a simple search bar, this one only allows searches for product units. Second are three buttons: Add product, Add unit and Update unit. These link to their respective pages, see below. Third is the user drop down menu. Here you can find links to various other sites, see below.




Above is the content of the drop-down menu. The links lead to different parts of the site as well as allowing the user to log out.





The add product page



On this page, you can add new products to the database. A product must have a CAS number and a (descriptive) name. You must also define the temperature range in which it should be kept and any constraints it might have, such as it being flammable, to ensure it is only stored in a compatible location.
You can also head straight to the add product unit page by clicking the "Add unit" link.





The unit CAS page



When the add unit button is clicked, it redirects you to this page. Here you must enter the CAS number of a product in the database. It will then take you to the add product unit where you can add a unit of that product to the database.





The add product unit page





Once the CAS number has been chosen, you get to the add product unit page. On this page you can add a new container of a particular chemical. You can change the product this unit is a part of, in case you chose the wrong CAS number. In order to create a new unit, you must input a description (we recommend something descriptive), a delivery date, a company that produced it, an initial amount as well as the measurement unit and finally a location.
All other attributes on this page are optional or have defaults but we recommend that you associate any product unit with at least one department. This allows users associated with this department to watch it's status, getting reminders if it is close to finished. If you intend to add several units with exactly the same attribute values, this is also possible. Simply input the number of units to add at the bottom field.





The add product unit page



If a product unit has been moved, it has been opened or some amount used, you can use the update product unit page to update the information for that unit. It is also possible to delete a unit from the database completely, though this is not advisable unless absolutely necessary. A unit can also be forcibly archived here, this normally only happens if it's current quantity drops to zero. The unit then disappears from the normal inventory but can still be found in the archive page, see drop-down menu above and page information below.





The reservations page



On the reservations page, you can see any reservations you have made and what day the reservation is made for. You can also access the pages to add new resrvations and update existing ones.





The add reservations page



Here, you can add new reservations on chemicals you will need to use. You will need to choose a specific product unit to reserve, the amount you need and when you intend to use it. You will also need to name the reservation.





The update reservations page



On this page, you can update an existing reservation with the amount used. It defaults to the reserved amount so if you used only that there is no need input a value for the amount.





The my inventory page



This page shows the inventory you are watching and does not include any product units that are not associated with a department you are associated with. It also has a link to the update unit page.





The running low page



Here, you can see an overview of all product units for which the current amount is running low. A product unit show up here once the current amount drops below 50% of the initial amount.





The running low page



This page shows an overview of all active product units in the database.





The running low page



On this page, you can see an overview of the locations currently available in the database. The page also includes links to pages for adding rooms and locations as well as for updating information on existing locations.





The add room page



On this page, you can add new rooms to the database. Each location must be associated with a room, as a given room might have several different types of locations, such as different refrigerators. A room must have a name and can optionally have a building name in case several buildings are being used.





The add location page



Here, you can add new locations to the database. Each location must be associated with a room and have a name. Optionally a description can be added. You then need to specify constraints which a product unit stored there can fulfill.





The update location page



This page allows you to update information on a location. You can either change the room or the description.





The archive page



Here, you can see an overview of all product units that have been archived. This automatically happens when a unit's current amount drops to zero but can also be done manually.





The add department page



This page allows you to add new departments to the database. Departments simply need to have a unique name. Departments are associated with users and product units. Each user associated with a given departments is also watching any product unit associated with that department, giving the reminders if a unit is running low and showing it in their inventory.





The my profile page



On this page, you can find information about the current user. There is also a link to the department association page which allows you to associate with a department.





The add association page



Here, you can associate the current user with new departments. This allows you to watch product units which the department is associated with.
